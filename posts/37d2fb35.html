MySQL语法使用(二)

发表于2024-09-20|更新于2024-11-04|使用教程数据库
|阅读量:

查看数据表

查看已创建的数据表

查看数据表结构

例: 查看student数据表的结构(USE命令可以让我们来使用数据库)
USE test;
DESC student;
或
USE test;
SHOW CREATE TABLE student;

数据表中数据的操纵

向数据表中添加数据

例: 向student数据表中添加一条记录(可只选择部分添加)
INSERT INTO student(sno, sname, age, sex, major, department)
VALUES(‘123’, ‘张三’, ‘23’, ‘男’, ‘计算机’, ‘信息学院’);

修改数据表中数据

例: 在student数据表中将张三年龄增加一岁
UPDATE student
SET age=age+1
WHERE sname=”张三”;

删除表中数据

例: 在student数据表中将张三数据删除
DELETE
FROM student
WHERE sname=”张三”;
